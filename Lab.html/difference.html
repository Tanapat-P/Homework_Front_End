Tag Difference
HR & BR เเท็ก HR เอาไว้ใช้สำหรับเส้นคั่นเเนวนอน
เเท็ก BR เอาไว้ใช้สำหรับขึ้นบรรทัดใหม่
ul & ol เเท็ก ul เอาไว้ใช้สำหรับลิสต์ที่เป็นสัญลักษณ์
เเท็ก ol เอาไว้ใช้สำหรับลิสต์ที่เป็นลำดับ
select & checkbox & radio button เเท็ก select เอาไว้ใช้สำหรับตัวเลือกรายการเเบบ dropdown
เเท็ก checkbox เอาไว้ใช้สำหรับตัวเลือกรายนการเเต่สามารถติ๊กพร้อมกันทั้งหมดได้
เเท็ก radio button เอาไว้ใช้สำหรับเตัวเลือกรายการเเต่สามารถติ๊กได้เเค่อันเดียว
div & span เเท็ก div เอาไว้ใช้สำหรับเเบ่งส่วนต่างๆใน document
เเท็ก span เอาไว้ใช้สำหรับจัดกลุ่มองค์ประกอบเเบบอินไลน์ในเอกสาร
script & style เเท็ก script เอาไว้ใช้สำหรับสำหรับการกำหนด javascript
เเท็ก style เอาไว้ใช้สำหรับ css
