Input Type
input type description
text field กล่องข้อความ
text area กล่องข้อความที่มีขนาดใหญ่
checkbox ช่องสี่เหลี่ยมเล็กๆเอาไว้ติ๊กข้อมูลที่ต้องการเเละติ๊กได้หลายอัน
radio button ช่องสี่เหลี่ยมเล็กๆเอาไว้ติ๊กข้อมูลที่ต้องการเเละติ๊กได้อันเดียว
list / menu จะมีลิสต์รายการ dropdown ลงมา
button จะเป็นปุ่มกด
hidden field ข้อความจะไม่โชว์บนบราว์เซอร์
